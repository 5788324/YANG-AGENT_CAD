# Antigravity 适配说明

本项目的核心规则不绑定单一 AI 客户端。Antigravity 接手时，应遵守：

1. 先阅读 `AGENTS.md`。
2. 再阅读 `docs/RUNTIME_CONTRACT.md`。
3. 当前状态看 `docs/PROJECT_STATE.md`。
4. 继续工作看 `handoff/CURRENT_HANDOFF.md`。
5. 每次结束必须更新 worklog 和 handoff。

## Antigravity 的角色

Antigravity 可以作为：

- 代码开发者
- 文档维护者
- 插件开发者
- 测试执行者
- 错误日志分析者

但不能绕过安全规则直接批量修改 DWG。
